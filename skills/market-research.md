# 의료기기 시장조사

의료기기 전용 정책펀드 용역의 **산업분석·타당성** 근거 데이터를 수집·정리하는 스킬. 시장규모·성장률·품목별
실적·업체수 등 정량 데이터를 모아 표+Excel로 낸다. 모든 데이터는 이 용역의 논리축에 봉사한다:
**"국내는 flat + death-valley 2회 + 경로의존성 → 그래서 정부 재원·전용펀드가 필요하다."**

## 핵심 원칙

- **연간 갱신 데이터는 반드시 재확인**. 시장규모·품목순위·업체수는 식약처/KHIDI가 매년 새로 발표한다.
  `references/market-data-sources.md`의 시드값은 마지막 확인 시점 참고치일 뿐, 표에 넣기 전 `web_search`로 최신 연도값을 확인하고 출처·기준연도를 명시한다.
- **논리축에 연결**. 숫자만 나열하지 말고 각 데이터가 "국내 flat / death-valley / AI 급성장 / 경로의존성" 중
  어느 논거를 뒷받침하는지 1줄로 연결한다.
- **무료 공식통계 우선**: 식약처 생산·수출·수입 실적 보도자료, KHIDI 보건산업통계·의료기기산업 분석. 글로벌은 유료
  리서치(GlobalData/Fitch BMI/Statista)가 많으므로, 유료면 수치 인용 시 출처·유료 여부를 표기하고 무료 대안(WHO·OECD·각국 규제기관)도 병기한다.
- **기준연도·단위 명확히**. "생산액 X억원(2025년, 식약처)"처럼 연도·출처·단위를 항상 붙인다. 환율 필요 시 기준 명시.

## 워크플로우

### 1단계 — 조사 범위 확정
요청에서 범위를 정한다: 국내 전체 / 글로벌 / 특정 품목(체외진단·영상·임플란트·전기수술기 등) / 특정 논거(AI 급성장만 등).
지정 없으면 국내 전체 + 글로벌 개요 + 4대 논거 데이터 세트를 기본으로 한다.

### 2단계 — 시드 로드
`references/market-data-sources.md`를 읽어 소스 목록, URL 패턴, 시드 참고값, 논거 매핑을 확보한다.

### 3단계 — 최신값 검증 (필수)
`web_search`로 최신 연도 데이터를 확인한다. 검색 예:
- `식약처 의료기기 생산 수출 수입 실적 <올해>`
- `KHIDI 의료기기산업 시장규모 CAGR`
- `global medical device market size <올해> forecast`
- `AI SaMD 소프트웨어 의료기기 허가 건수 식약처`
확인된 수치만 표에 넣고, 각 행에 출처 URL + 기준연도를 붙인다. 못 찾은 값은 "확인 필요".

### 4단계 — 출력 (표 + Excel)
**(a) 채팅 표**: 아래 템플릿. 국내/글로벌/품목별/논거별 섹션.
**(b) Excel**: 행 데이터를 JSON으로 만들어 `scripts/build_workbook.py`로 .xlsx 생성.

```bash
cd <스킬경로>
python scripts/build_workbook.py --rows /tmp/market_rows.json --out /mnt/user-data/outputs/의료기기_시장데이터.xlsx
```

JSON 스키마는 `scripts/build_workbook.py` docstring 참조(category, metric, value, unit, year, region, source_org, url, thesis, note). 생성 후 `present_files`로 제시.

```
## 의료기기 시장 데이터

...

```

## 주의
- 시장규모·순위를 시드값 그대로 쓰지 말 것. 반드시 최신 연도 재확인(3단계).
- 글로벌 수치는 리서치사마다 정의(대상 범위)가 달라 편차가 크다. 출처·정의를 함께 적고 단일 수치로 단정하지 않는다.
- 펀드 구조·GP·거버넌스·해외펀드 사례 요청은 meddevice-fund-benchmark 스킬로 넘긴다. 여기선 시장·산업 데이터만.
